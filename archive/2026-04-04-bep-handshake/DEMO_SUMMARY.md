# Syncthing Rust 演示总结

**演示程序**: `demo.exe`  
**运行方式**: `demo --mode interactive` (交互) 或 `demo --mode quick` (快速)

---

## 演示功能概览

### 1. 📋 核心类型演示
- **DeviceId**: 设备唯一标识 (SHA-256)
- **VersionVector**: 版本向量冲突检测
- **冲突解决**: 自动检测并发修改

### 2. 📁 文件系统演示  
- **文件扫描**: 进度条展示
- **块哈希**: SHA-256 计算
- **忽略模式**: .stignore 规则匹配

### 3. 💾 数据存储演示
- **分层存储**: Cache + Disk
- **内容寻址**: 哈希去重
- **快速读取**: 缓存命中

### 4. 🌐 网络层演示
- **Iroh P2P**: QUIC + NAT穿透 + 中继
- **连接管理**: 池化 + 保活 + 重连
- **安全**: TLS 1.3 加密

### 5. 🔄 同步引擎演示
- **增量同步**: 仅传输差异
- **冲突解决**: 多策略支持
- **版本控制**: 向量时钟

### 6. 📡 API 演示
- **REST API**: 完整端点列表
- **WebSocket**: 实时事件流
- **配置管理**: 动态配置

### 7. 🎯 完整工作流
- 10步完整同步流程演示
- 实时进度展示
- 同步统计报告
